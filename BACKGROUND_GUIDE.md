## 📝 **如何使用**

### **方法 1: 直接使用图片链接**

1. 访问 Unsplash
2. 搜索 "dark aesthetic background"
3. 选择喜欢的图片
4. 点击 "Download free"
5. 复制图片链接
6. 告诉我链接，我帮你配置

### **方法 2: 上传到 GitHub**

1. 下载图片
2. 放到博客仓库的 `images/` 目录
3. 我帮你配置使用

## 🎯 **配置示例**

### **Jekyll Minimal 主题**

在 `css/` 目录创建自定义样式：

```css
/* 自定义背景 */
body {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  background-attachment: fixed;
}

/* 或使用图片 */
body {
  background-image: url('/images/background.jpg');
  background-size: cover;
  background-attachment: fixed;
  background-position: center;
}
```
