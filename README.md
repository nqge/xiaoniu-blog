## 如何部署

### 方法一：GitHub Pages（推荐）

1. Fork 这个仓库
2. 进入 Settings → Pages
3. Source 选择 `Deploy from a branch`
4. Branch 选择 `main`，文件夹选择 `/root`
5. 访问 `https://你的用户名.github.io/xiaoniu-blog/`

### 方法二：本地预览

```bash
# 安装 Jekyll
gem install bundler jekyll

# 预览
bundle exec jekyll serve

# 访问 http://localhost:4000
```
